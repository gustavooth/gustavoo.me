Sobre Mim

Motivado a entender como as coisas funcionam, iniciei minha jornada na programação pelos fundamentos. Ao aprender C, desenvolvi uma base sólida em lógica de programação, algoritmos e gestão de memória, o que facilitou o aprendizado de linguagens de mais alto nível.

Hoje, combino essa base técnica com ferramentas modernas para criar soluções completas. No ecossistema JavaScript e TypeScript, desenvolvo interfaces dinâmicas e aplicações escaláveis utilizando React e Next.js, entregando a melhor experiência para o usuário.

No back-end, aplico Python para otimizar processos através de scripts de automação e tratamento de dados. Com Java, emprego conceitos sólidos de Orientação a Objetos para garantir a robustez de aplicações, atuando tanto em novas features quanto na manutenção de serviços legados.
